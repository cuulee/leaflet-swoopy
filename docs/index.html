Leaflet Swoopy

A plugin for adding beautiful swoopy arrows on top of Leaflet maps

Installation

npm install leaflet-swoopy

Usage Example

import 'leaflet-swoopy';

// create leaflet map ...

const swoopy = L.swoopyArrow([53.52, 13.4], [53.525, 14.41], {
  annotation: 'Hi!',
  fontSize: 12,
  iconAnchor: [20, 10],
  iconSize: [20, 16]
}).addTo(map);
