Leaflet Swoopy

A plugin for adding beautiful swoopy arrows on top of Leaflet maps

Installation

npm install leaflet-swoopy

Usage Example

new L.SwoopyArrow([53.52, 13.4], [53.525, 14.405], {
  htmlLabel: 'Yo',
  fontSize: 12,
  iconAnchor: [20, 10],
  iconSize: [20, 16]
}).addTo(map);
Option Type Default Description 
htmlLabel
 
String
 
null
 Text label at the fromLngLat position
